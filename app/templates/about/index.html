Help & About

What is WhatSend?

WhatSend lets you schedule WhatsApp messages to be sent immediately or at a specific time — one-time or recurring. You create a job with a message, pick a group, and choose a trigger — the app handles the rest.

The story

Originally named WhatSend, this project was born out of a simple problem: we had a place where WhatsApp messages needed to be sent for events on specific days, but we'd often forget or be late.

v1 was a Python script using pywhatkit to send messages automatically. It worked — kinda. But it required a browser with an already signed-in WhatsApp user, was painfully slow, and couldn't run on a server.

We went back to manual sending for a while. Then v2 appeared: same idea but powered by the Whapi.Cloud API instead of pywhatkit. A huge improvement — no browser needed, fast, server-ready.

But it was still configured via a YAML file, which meant editing files and redeploying every time something changed. So v3 added REST APIs so everything could be configured remotely. Then the question came: how will normal people use this? That's when the web dashboard was born — Jinja2 templates, HTMX for interactivity, zero JavaScript build step.

And that's where we are today. WhatSend is a full-featured WhatsApp scheduler you can deploy on a $5/month server and control from any browser.

Getting a Whapi.Cloud token

  1. Go to whapi.cloud and sign up.
  2. Create a new channel and link your WhatsApp account by scanning the QR code.
  3. Copy your API token from the channel dashboard.
Note: The token is a long string like a1b2c3d4e5f6.... You'll paste this into WhatSend when adding a token.

How to use

1. Add an API token

Go to Tokens → + Add Token, give it a name (e.g. "work"), and paste your Whapi.Cloud API token.

2. Create a job

Go to Jobs → + New Job. Fill in:

  • API Token — which token to send with
  • Label — a friendly name for your own reference
  • Group — select from your fetched WhatsApp groups, or enter a group ID manually
  • Message — the text to send (emoji supported ✅)
  • Image — optional image attachment (JPEG/PNG/GIF/WebP, max 5MB)
  • Trigger — Send Now (fires immediately), One-time (pick date & time), or Recurring (Daily / Weekdays / Weekly / Monthly)
Tip: Times are converted from your timezone (shown in the nav bar) to UTC automatically. Click the timezone badge to change it.

3. Monitor

Check the Logs page to see if your messages were sent successfully. Click Show response to view the full API response.

Your WhatsApp number must be in the group

Whapi.Cloud sends messages through your linked WhatsApp number. That number must be a member of any group you want to send to. If you get a 403 error, make sure your WhatsApp account has been added to the group.

Timezone

You set your timezone during signup. It's stored in your account and used to convert dates between your local time and UTC. Click the timezone badge in the nav bar to change it at any time.
