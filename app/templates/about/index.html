Help & About

What is WhatSend?

WhatSend lets you schedule WhatsApp messages to be sent at a specific time or on a recurring schedule. You create a job with a message, a group ID, and a trigger — and the app handles the rest.

Getting a Whapi.Cloud token

  1. Go to whapi.cloud and sign up.
  2. Create a new channel and link your WhatsApp account by scanning the QR code.
  3. Copy your API token from the channel dashboard.
Note: The token is a long string like a1b2c3d4e5f6.... You'll paste this into WhatSend when adding a token.

How to use

1. Add an API token

Go to Tokens → + Add Token, give it a name (e.g. "work"), and paste your Whapi.Cloud API token.

2. Find a group ID

Group IDs look like 120363xxxxxxxxxxxxxxxxxx@g.us. To find yours:

  • Open WhatsApp Web, go to the group chat
  • Or use the Whapi.Cloud API to list your chats

3. Create a job

Go to Jobs → + New Job. Fill in:

  • API Token — which token to send with
  • Label — a friendly name for your own reference
  • Group ID — the WhatsApp group to send to
  • Message — the text to send (emoji supported ✅)
  • Image path — optional path to an image file on the server
  • Trigger type — one-time (date) or recurring (cron)
  • Trigger value — YYYY-MM-DD HH:MM for dates, or a cron expression like 0 9 * * 1 for recurring (minute hour day month weekday)
Tip: All times are in UTC. Adjust accordingly if you're in a different timezone.

4. Monitor

Check the Logs page to see if your messages were sent successfully.

CronMeaning

Original script (CLI)

This project started as a simple Python script that reads a config.yaml file. If you prefer the terminal, the original scheduler.py and sender.py are still included and work independently.
